# Deployment and Cloud Infrastructure

## 1. What is Argo CD used for?
- GitOps deployment tool
- Primary functions:
  - Configuration synchronization
  - Automated deployments
  - Not used for logging purposes

## 2. What is Azure Red Hat OpenShift (ARO)?
- Joint offering from Microsoft Azure and Red Hat
- Managed OpenShift service features:
  - Automatic scaling
  - Security patch management
  - Cluster upgrade automation
  - Azure Monitor integration

## 3. What is OCP4?
- OpenShift Container Platform 4
- Features:
  - Kubernetes-based platform
  - Container orchestration
  - Microservices management

## 4. How does Azure integrate with OCP4?
- Provides managed OpenShift environment
- Runs within Azure cloud infrastructure
- Combines:
  - Azure's cloud capabilities
  - OpenShift's container platform features
