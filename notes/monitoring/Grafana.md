# Monitoring and Observability Tools

## 1. What is Grafana?
- Dashboard and visualization tool
- Features:
  - Connects to multiple data sources:
    - Prometheus
    - Elasticsearch
    - Azure Monitor
  - Creates visual dashboards
  - Configurable alerting system
    - Email notifications
    - Slack integration
    - Teams channel alerts

## 2. What is Elasticsearch?
- Search and analytics database
- Primary uses:
  - Log storage
  - Event tracking
  - Text data analytics at scale
- Visualization:
  - Compatible with Kibana
  - Supports Grafana integration

## 3. What is Prometheus?
- Time-series metrics collection system
- Monitors:
  - CPU usage
  - Memory utilization
  - Network performance
  - Application metrics
- Kubernetes/OpenShift integration
- Primary data source for Grafana dashboards
